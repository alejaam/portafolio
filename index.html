Mi enfoque actual

Trabajo principalmente con React y TypeScript, utilizando Next.js para aplicaciones que requieren renderizado del lado del servidor o generación de sitios estáticos. En el frontend, me enfoco en crear interfaces accesibles y responsivas con Chakra UI y Tailwind CSS.

En el backend, prefiero Node.js con Express o NestJS para construir APIs RESTful, utilizando PostgreSQL o MongoDB según los requisitos del proyecto.

Para el despliegue, trabajo con Vercel para frontend y Render o Railway para servicios backend, implementando CI/CD con GitHub Actions.
